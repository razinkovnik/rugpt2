
Химики из Санкт-Петербургского университета ИТМО разработали первый в мире способ печати голографических изображений с помощью струйного принтера. В качестве чернил ученые использовали наночастицы диоксида титана. Результаты работы опубликованы в Advanced Functional Materials.Ученые синтезировали наночастицы диоксида титана с помощью кислотного гидролиза изопропилата титана. Исследователи смешали полученный золь частиц с этанолом для изменения поверхностного натяжения и вязкости системы. Далее по прошествии двенадцати дней, в течение которых происходит полная гомогенизация системы, команда удалила избыток растворителя с помощью роторного испарителя. В результате образовались прозрачные чернила, обладающие оптимальными вязкостью и содержанием твердой фазы, наилучшим образом подходящими для печати.Для печати исследователи взяли обычный настольный принтер Canon Pixma IP 2840, в качестве подложки для печати использовали коммерческие полиэтилентерефталатные пленки, помещенные на листы A4 обычной бумаги, а чернила из диоксида титана залили в стандартный картридж для принтера. Толщина полученного в результате такой печати слоя диоксида титана не превышает 500 нанометров. Для защиты напечатанного изображения исследователи покрыли бумагу прозрачным слоем полиизобутилметакрилата толщиной около четырех микрон.
