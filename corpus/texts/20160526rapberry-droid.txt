
В репозитории AOSP (Android Open Source Project) появилась отдельная ветка для одноплатного компьютера Raspberry Pi 3. Это, скорее всего, означает, что в будущем выйдет официальная версия Android для Raspberry Pi 3, что позволит использовать на одноплатном компьютере различные приложения из Google Play, отмечает ArsTechnica.Кроме Android на Raspberry Pi можно устанавливать несколько дистрибутивов Linux и Windows 10 IoT Core — для последней операционной системы даже существуют готовые программно-аппаратные комплекты.Raspberry Pi 3 заменил предыдущее поколение популярных микрокомпьютеров в феврале 2016 года. Raspberry Pi 3 Model B по сравнению с Pi 2 Model B обладает 64-разрядным четырехъядерным процессором ARMv8 и встроенной поддержкой Wi-Fi 802.11n, Bluetooth 4.1 и BLE. Официальная цена устройства составляет 35 долларов.Компьютер Raspberry Pi часто используется в различных «гаражных» проектах изобретателями и конструкторами. При необходимости расширить функциональность компьютера к нему можно подключить периферийное оборудование по USB. Также в октябре 2015 года стало известно, что Raspberry Pi можно заказать по индивидуальному проекту, однако в одной партии должно быть от трех тысяч компьютеров.Николай Воронцов
