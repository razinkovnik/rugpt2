
Исследователи из Лаборатории информационных технологий и искусственного интеллекта Массачусетского технологического института представили технологию позиционирования при помощи Wi-Fi передатчиков потребительского уровня. По словам разработчиков, подобная технология может быть использована в том числе для избирательного предоставления доступа без пароля к Wi-Fi сетям — например, в кафе. Подробнее о новой технологии сообщает IEEE Spectrum.Технология под названием Chronos использует два устройства для определения расстояния между ними. При работе передатчик и приемник синхронно переключаются между 35 разными полосами частот от 2,4 до 5,8 гигагерц. Частота меняется каждые две-три микросекунды и принимающая сторона по разнице времени получения волн от передатчика может вычислить расстояние, которое прошли радиоволны. Если хотя бы одно из устройств при этом имеет больше одной антенны, то при помощи триангуляции можно вычислить не только расстояние, но и точно местоположение второго устройства. По словам разработчиков, Chronos позволяет определить расстояние между объектами с точностью до нескольких сантиметров. Подобную технологию Wi-Fi-позиционирования кроме можно использовать для избирательного подключения клиентских устройств к сети, навигации внутри помещений или, например, в робототехнике для определения приближения человека по его мобильному телефону.Ранее другая группа исследователей представила метод непрерывной передачи данных между компьютерами при помощи ультразвука. Используя динамики и микрофон обыкновенного ноутбука, авторам удалось передавать данные со скоростью 200 бит в секунду при точности 90 процентов. По словам ученых, такой подход также можно использовать для передачи паролей от Wi-Fi сети в пределах помещения.В MIT ранее уже демонстрировали альтернативное применение Wi-Fi — весной 2015 года разработчики приспособили роутер для дистанционного измерения пульса и частоты дыхания человека. Кроме того, ученые из Калифорнийского университета в Санта-Барбаре научились при помощи Wi-Fi роутеров считать пешеходов.Николай Воронцов
