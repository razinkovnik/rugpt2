
Институт роботов и «умных» машин Технологического института Джорджии разработал новый алгоритм безопасности, который позволяет большой группе роботов выполнять задания не сталкиваясь друг с другом и не мешая друг другу. Согласно сообщению института, алгоритм основан на уменьшении зоны безопасности роботов, небольшого пространства вокруг аппаратов, в которое не должны попадать другие аппараты.Групповое выполнение задания роботами имеет серьезный недостаток. Каждый из аппаратов в группе обычно оснащается собственной программой, подчиненной алгоритмам безопасности, которые создают вокруг роботов зоны безопасности. Выполняя задание, роботы стараются передвигаться таким образом, чтобы в их зону безопасности не попадали ни посторонние предметы, ни другие роботы.Когда роботы выполняют какое-либо задание небольшой группой, трудностей обычно не возникает — им хватает свободного пространства для перемещения. В случае крупной группы могут возникать ситуации, когда роботы начинают мешать друг другу или вовсе становятся не способными выполнять задание, потому что при каждом движении один из аппаратов обязательно попадает в зону безопасности другого.Ученым из Технологического института Джорджии удалось преодолеть этот недостаток, хотя подробности о новом алгоритме пока не раскрываются. Детально свою работу исследователи намерены представить на конференции IEEE CDC, которая пройдет в Лас-Вегасе 12–14 декабря текущего года. Лабораторные испытания алгоритма проводились в группе из четырех и восьми роботов, а также в группе из четырех аппаратов, один из которых не соблюдал зоны безопасности.Во всех случаях роботы были выстроены по углам комнаты. Им было дано задание поменяться местами, проехав через центр комнаты. Роботы из групп, где каждый член был оснащен новым алгоритмом, доезжали до центра комнаты, объезжали его по кругу и занимали свои новые места. В группе с роботом, не соблюдающим правила безопасности, три аппарата сначала пропустили его, а затем завершили маневр.В контрольных испытаниях роботов с обычными алгоритмами безопасности либо происходило столкновение, либо аппараты прекращали выполнение задания. По мнению разработчиков, новые алгоритмы безопасности можно будет использовать в беспилотных автомобилях или перспективных пассажирских самолетах.Сегодня сразу несколько компаний заняты разработкой роботов, способных действовать группами. Каждый аппарат будет отвечать за выполнение одного конкретного действия, а группа при этом сможет выполнить задание лучше, чем один универсальный робот. Считается, что это позволит снизить стоимость каждого робота в группе и увеличить шансы на выполнение заданий.Групповое выполнение заданий, например, лежит в основе концепции Пентагона Gremlins. Этот проект предполагает создание множества дешевых беспилотников, которые можно будет сбрасывать стаями с транспортных самолетов. Они будут группами вести разведку или выводить из строя радиолокационные станции.Василий Сычёв
