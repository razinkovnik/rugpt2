
Компания Smart 3D представила прототип платформы для создания 3D-моделей небольших объектов при помощи смартфона. В октябре запланирован запуск краудфандинговой кампании по сбору средств на производство сканера Pixelio, говорится на сайте компании.Устройство представляет собой платформу с вращающимся вокруг нее и объекта сканирования креплением для смартфона. При вращении телефона платформа, связанная с телефоном по Bluetooth, с нужной частотой делает снимки, из которых путем 3D-реконструкции будет сформирована трехмерная модель объекта. Для создания 3D-модели приложение Pixelio передает данные в бесплатное приложение Autodesk 123D Catch.Для смартфонов, не поддерживающих связь по Bluetooth 4.0, компания обещает делать снимки при помощи технологии Virtual Finger, в применении которой каким-то образом участвует графен, других подробностей о Virtual Finger не сообщается.
