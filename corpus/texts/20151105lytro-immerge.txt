
Компания Lytro представила панорамную пленоптическую камеру Lytro Immerge. По словам представителей компании, это первая коммерческая панорамная камера светового поля. Подробнее о Immerge можно прочитать на сайте компании.Как объясняют представители компании, их цель — разработка готового набора инструментов для работы с контентом, записанным при помощи Immerge. Для обработки и хранения данных с камеры компания предлагает использовать выделенный сервер, а для воспроизведения контента специально разработанное программное обеспечение, которое поддерживает практически все современные VR-очки.Lytro Immerge для каждой точки изображения записывает информацию не только о цвете и интенсивности света, но также векторную информацию о направлении света. Таким образом, для человека, осматривающегося в записанной подобным образом виртуальной реальности, изображение будет выглядеть крайне реалистичным.Одна из особенностей пленоптических камер — возможность перефокусировки сделанного изображения. Таким образом, отмечают представители Lytro, панорамная камера светового поля переводит создание VR-контента на новый уровень, позволяя решить одну из актуальных проблем виртуальной реальности — отсутствие низкой глубины резкости в точке, куда смотрят глаза (ранее одно из решений этой проблемы путем доработки VR-очков предложили исследователи из Стэнфордского университета).
