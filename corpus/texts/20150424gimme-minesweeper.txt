
Microsoft включила
«Солитер» в дистрибутив Windows 10, сообщает издание The Verge.Стандартные игры исключены из дистрибутивов операционных систем Microsoft начиная с Windows 8, хотя по-прежнему были доступны для установки через интернет. «Эти игры не устанавливаются в Windows 8 автоматически, но их можно бесплатно скачать из Магазина Windows», — сообщается на сайте Microsoft.
