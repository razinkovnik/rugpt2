
Яндекс станет поисковой системой по умолчанию для операционной системы Microsoft Windows 10 на территории России, Белоруссии, Казахстана, Украины и Турции. Об этом сообщает Интерфакс.В рамках сотрудничества Яндекса и Microsoft поисковик Bing, установленный по умолчанию в браузерах Microsoft Edge и Internet Explorer на Windows 10, будет заменен на аналогичный сервис Яндекса. Кроме того, Яндекс займется разработкой приложений для Windows, среди которых будут программы для для заказа такси, поиска информации, картографические приложения и навигатор, музыкальный проигрыватель и приложение для поиска товаров. По-видимому, речь идет о разработке выделенных windows-приложений для использования собственных сервисов Яндекса.Windows 10 вышла 29 июля 2015 года в 190 странах на 111 языках. Пользователи, зарезервировавшие заранее копию для обновления с Windows 7 и 8, получили бесплатное обновление операционной системы. Активированную версию Windows 10 также удалось получить после обновления некоторым пользователям пиратских версий Windows.Николай Воронцов
