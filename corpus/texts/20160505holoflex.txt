
Исследователи из Human Media Lab Королевского университета в Кингстоне разработали гибкий смартфон с дисплеем светового поля, который может выводить трехмерное изображение, доступное для просмотра невооруженным глазом сразу нескольким пользователям. Авторы разработки представят доклад на конференции ACM CHI 2016, кратко о гибком голографическом смартфоне HoloFlex можно прочитать на сайте лаборатории.
Экран голографического смартфона представляет собой 3D-печатный массив из 16 тысяч миниатюрных сверхширокоугольных объективов, каждый из которых отвечает за вывод сегмента изображения диаметров в 12 пикселей. В итоге экран с изначальным разрешением 1920×1080 пикселей способен выводил объемное изображение светового поля разрешением 160×104. Изображение выглядит объемным независимо от угла обзора и не требует какого-либо дополнительного оборудования для просмотра.
Внутри гибкого голографического смартфона установлен 1,5-гигагерцовый процессор Qualcomm Snapdragon 810 и два гигабайта оперативной памяти. Работает гаджет под управлением операционной системы Android 5.1. Для управления используются обычные жесты для тачскрина, а жесты «в глубину» осуществляются при помощи сгибания смартфона. 
По словам разработчиков, использование оси Z в управлении гаджетом позволяет, например, редактировать с помощью смартфона трехмерные модели. В опубликованном видео авторы демонстрируют редактирование модели чайника, — судя по внешнему виду, в качестве примера используется модель «Чайник Юта» — и физическое сгибании смартфона вызывает смещение ручки модели в глубину рабочего пространства.
Ранее специалисты из Human Media Lab уже разрабатывали гнущиеся мобильные устройства смартфонами. В феврале 2016 года исследователи представили гибкий смартфон с OLED-экраном, который также управлялся при помощи физического сгибания корпуса.
