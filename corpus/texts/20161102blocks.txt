
Британская компания Roli выпустила модульный сенсорный синтезатор Blocks, который позволяет пользователям с разным уровнем подготовки создавать собственные музыкальные треки. Кратко о новом устройстве рассказывает The Next Web.Синтезатор Blocks состоит из модулей, соединяющихся друг с другом с помощью магнитных креплений, однако базовый модуль может работать и самостоятельно. Базовый модуль Lightpad Block представляет собой чувствительный к нажатиям квадратный блок с 225 светодиодами. С помощью этого модуля пользователь может создавать собственные треки, используя нажатия, «свайпы» и другие жесты. Дополнительные модули Live Block и Loop Block оснащены дополнительными кнопками, которые облегчают взаимодействие с треками и семплами.Разработчики делают основной упор на портативность и интуитивность управления гаджетом — владелец Blocks может создавать музыку, не обладая опытом работы с аудиофайлами или навыками игры на музыкальных инструментах. При этом Blocks также подходит для профессионалов, которые могут управлять большим количеством разных модулей, настроенных под конкретные нужды, — для настройки звучания модулей используется приложение-компаньон под iOS.Концепция модулей Blocks напоминает модульный контроллер Palette, который позиционируется как устройство для работы в музыкальных и графических редакторах. При этом Palette выглядит более профессиональным инструментом, поскольку позволяет комбинировать отдельные кнопки, слайдеры и валкодеры в зависимости от потребностей пользователя, в то время как Blocks предоставляет упрощенный интерфейс, направленный на интуитивное создание музыки, и не предполагает использования в других сферах.Николай Воронцов
