
Американские ученые запустили онлайновый сервис для анализа
интернет-мемов. Препринт его описания приводится в PeerJ Preprints.
Разработка сотрудников Университета Индианы называется The
IUNI Observatory on Social Media, или сокращенно OSoMe (читается как английское
слово awesome — «потрясающий»). Как
говорится в пресс-релизе университета, сервис для анализа «онлайновых
тенденций, мемов и других вспышек вирусной активности» предназначен для «журналистов,
исследователей и широкой публики».
OSoMe, разработанный на основе пакета обработки больших
объемов данных Apache Big Data Stack (ABDS), включает следующие инструменты: Trends для изучения динамики
популярности мемов; Movies для создания видео, иллюстрирующих, как обсуждение мемов меняется
со временем; Networks для анализа того, кто обсуждает мемы, и как эти мемы
взаимосвязаны; Maps для изучения географии распространения мемов.
Кроме того, сервис включает предыдущую разработку лаборатории,
которая называется BotOrNot и предназначена для обнаружения ботов в Twitter, а также интерфейс приложения (API),
позволяющий сторонним исследователям надстраивать систему для выполнения
дополнительных задач.
Материалом для анализа служат хэштеги Twitter, количество сообщений в котором
превысило 70 миллиардов. Сервис позволяет агрегировать данные по нескольким
хэштегам одновременно.
