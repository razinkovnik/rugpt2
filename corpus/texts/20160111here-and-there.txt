
Специалисты Google, работающие над развитием AR-технологии Project Tango, продемонстрировали работу приложения, измеряющего трехмерные объекты в реальном времени. Видео с демонстрацией работы приложения опубликовано на YouTube-канале Project Tango.В опубликованном видео сотрудник Google использует мобильное устройство как линейку для измерения объектов и расстояний в окружающем пространстве. Полный список приложений для устройств с технологией Project Tango доступен для загрузки в Google Play.Project Tango работает над технологией использования мобильных устройств с датчиком глубины для взаимодействия с окружающим миром. Мобильные устройства Project Tango способны отслеживать перемещение и строить трехмерную карту окружающего мира в режиме реального времени. Ранее на выставке CES 2016 сообщалось, что компания Lenovo запустит производство 6,5-дюймовых смартфонов с поддержкой технологии Project Tango.Николай Воронцов
