
В стандарт HTTP официально включен новый код состояния, который предлагается использовать для заблокированных по какой-либо причине страниц. В качестве числа для нового кода состояния выбрано 451 — отсылка роману Рэя Брэдбери «451 градус по Фаренгейту». Об этом сообщил в своем блоге глава Инженерного совета интернета (IETF).Новый код состояния входит в класс кодов, которые сообщают об ошибке при запросе со стороны клиента. Сервер ответит на запрос кодом 451, если запрашиваемый контент недоступен из-за требований правообладателя или запрещен правительством страны, в которой находится клиент.Изначально «код цензуры» был предложен в 2013 году. Как объяснил глава IETF, новый код поможет точнее определить количество заблокированного по различным причинам контента в интернете. Кроме того, код 451 будет понятнее для пользователя, поскольку при использовании кодов 403, 404 и 410 причины невозможности доступа не сообщаются.Коды состояний используются в протоколе HTTP для краткого цифрового ответа сервера на запрос клиента. Первая цифра трехбуквенного числа указывает на один из пяти классов кода, некоторые из которых не отображаются непосредственно пользователю. Наиболее часто рядовые пользователи видят коды ошибок: 403 (доступ запрещен), 404 (запрашиваемый ресурс не найден) и 502 (ошибка шлюза).  Николай Воронцов
