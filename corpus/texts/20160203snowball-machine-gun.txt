
Инженер NASA Марк Робер сделал из машины для уборки опавшей листвы пулемет, который стреляет снежками. Инструкцию по созданию инженер выложил в свободном доступе.Снежковый пулемет сделан из машины для уборки листвы с помощью дополнительной трубы, скоб, изоленты и тройника трубопровода. Для того чтобы пулемет стрелял сразу большим количеством снежков, он снабдил его магазином, сделанным из трубы, в который помещается заряд из 15 небольших снежков. Магазин можно быстро отсоединить и вставить новый, что обеспечивает непрерывность стрельбы. Скорострельность снежкового пулемета регулируется с помощью интенсивности воздушного потока машины. За полсекунды оружие может выпустить очередь из 13 снежков.Ранее швейцарец Кевин Томас создал с помощью 3D принтера турель, которая способна самостоятельно обнаруживать движущиеся цели и открывать по ним огонь канцелярскими резинками.Кристина Уласович
