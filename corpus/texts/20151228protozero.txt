
Создатель блога Average Man vs Raspberry Pi разработал макетную плату ProtoZero для микрокомпьютера Raspberry Pi Zerо. Проект собрал необходимые для производства средства при помощи краудфандинга. Об этом сообщается на странице проекта на Kickstarter.ProtoZero представляет собой макетную плату размером с Raspberry Pi Zero, которая присоединяется к 40-контактному разъему расширенного ввода-вывода (GPIO), образуя «этажерку». По словам разработчика платы, подобный формат удобнее для создания и отладки различных процессов, при этом сопоставимые размеры платы позволяют использовать ее в тех проектах, где критичен размер. Автор проекта отмечает, что ProtoZero можно соединять в несколько уровней. Кроме того, плату можно использовать и с другими компьютерами семейства Raspberry Pi.
