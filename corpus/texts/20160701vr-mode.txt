
Очередная бета-версия и версия для разработчиков Google Chrome для операционной системы Android указывают на разработку технологии виртуальной реальности для браузера. Об этом сообщает портал RoadToVR.В бета-версии появились настройки WebVR — API, позволяющий просматривать в VR-очках сайт, которые не были специально спроектированы для виртуальной реальности. В версии для разработчиков обнаружилась функция VR Shell, которая в перспективе позволит использовать браузер в очках виртуальной реальности типа Google Cardboard, однако в данной версии функция еще неработоспособна.Ранее подобную функциональность реализовала Samsung в браузере для очков виртуальной реальности Gear VR. Для ввода информации приложение использует распознавание голоса, также пользователь может для управления воспользоваться «режимом взгляда», в котором пункты меню выбираются при помощи технологии отслеживания взгляда.Николай Воронцов
