
Mozilla продолжит выпускать обновления безопасности браузера Firefox для Windows XP и Vista вплоть до сентября 2017 года. Об этом сообщается на сайте Mozilla.Microsoft со временем прекращает поддержку устаревших операционных систем, и сторонние разработчики, как правило, через некоторое время перестают обновлять программы, выпущенные для старых версий Windows. Тем не менее, Windows XP была широко распространенной операционной системой, поэтому многие разработчики, включая Mozilla, длительное время поддерживали обновление соответствующих версий своего программного обеспечения. Сама Microsoft официально прекратила поддержку Windows XP для частных пользователей в апреле 2014 года. Поскольку критические обновления безопасности больше не выпускаются, компания рекомендует пользователям переходить на современные версии Windows. Расширенная поддержка Windows Vista продлится до 11 апреля 2017 года. В марте 2017 года пользователи Firefox на операционных системах Windows XP и Vista будут переведены на релиз с расширенной поддержкой ESR. Предварительно планируется выпускать обновления безопасности до сентября 2017 года, однако в середине года Mozilla планирует оценить количество оставшихся пользователей браузера на устаревших операционных системах для точного определения даты завершения поддержки.Другой популярный браузер, Google Chrome, изначально поддерживал обновления под Windows XP только до апреля 2015 года, однако затем разработчики продлили поддержку Chrome для Windows XP до конца 2015 года.Николай Воронцов
