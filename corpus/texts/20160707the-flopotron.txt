
Программист Павел Задрожняк (Paweł Zadrożniak) сконструировал из дисководов, жестких дисков и сканеров музыкальную установку, которую назвал «Флопотрон 2.0». С помощью флопотрона автор исполнил песню Smells Like Teen Spirit группы Nirvana, главную тему из «Звездных войн» и главную тему из телесериала «Гавайи 5-O».Для постройки проекта разработчик использовал 64 дисковода, 8 жестких дисков и два оптических сканера. Как отмечает автор, в принципе катушки жестких дисков тоже можно было бы использовать в качестве импровизированных динамиков, но это, по его словам, было бы слишком легко, поэтому жесткие диски используются для имитации ударных. Каждая колонна из восьми дисководов управляется контроллером ATMega16 и отвечает за один канал звука, а количество задействованных дисководов влияет на громкость. Для управления сканерами автор использовал Arduino Uno. Управляет оркестром программа на Python, которая также может конвертировать MIDI-файлы для исполнения на дисководах, что, как отмечает изобретатель, упрощает процесс портирования песен для флопотрона.Это не первый эксперимент автора с воспроизведением музыки с помощью флоппи-дисководов. В 2011 году он с помощью двух дисководов исполнил «Имперский марш» из «Звездных войн».Флопотрон — один из самых больших массивов дисководов, использующийся для воспроизведения музыки. Ранее молодежный кружок в Германии собрал орган из 49 флоппи-дисководов с помощью частей, распечатанных на 3D-принтере. Николай Воронцов
