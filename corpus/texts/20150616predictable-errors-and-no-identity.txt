
Пользователь ресурса GitHub под ником stuartpb обнаружил, что популярные браузеры Google Chrome, Mozilla Firefox и Safari из-за программной недоработки могут вызвать зависание компьютера. Об этом сообщает СNews.Зависание компьютера может возникнуть из-за того, что браузеры не проверяют размер файла favicon.ico перед загрузкой. Таким образом, если программа получает файл, превышающий по размеру объем доступной для использования оперативной памяти, то компьютер перестает реагировать на команды пользователя. По словам самого stuartpb, на исследование проблемы его натолкнул твит о загрузке файла иконки размером в 64 мегабайта.На данный момент ошибка обнаружена на OS X 10.10 Yosemite, для Windows и Linux тесты не проводились. Созданы багрепорты для Firefox и Chrome.
