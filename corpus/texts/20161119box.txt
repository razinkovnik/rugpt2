
Американская компания AR15 Build Box разработал специальную коробку, с помощью которой начинающие стрелки смогут собрать свою самозарядную винтовку AR-15 и не забыть купить необходимые для этого детали. Коробка получила название Build Box. Ее стоимость составляет 18 долларов не включая стоимость доставки.Начинающие стрелки, желающие самостоятельно собрать оружие, обычно не помнят список всех запчастей, которые необходимы для этого. Из-за этого они нередко заказав и забрав в магазине один комплект деталей вскоре возвращаются, чтобы докупить еще несколько запчастей. При этом консультанты в оружейных магазинах не всегда могут подсказать, какие именно детали нужны для сборки.Build Box является своего рода памяткой для человека, желающего собрать собственную винтовку AR-15. Она представляет собой картонную коробку, в которую вложен минимальный список деталей, необходимых для сборки оружия, а также держатель из вспененного полиэтилена. В этом держателе выполнены прорези для деталей, необходимых для сборки.В держателе выполнены прорези для верхней части ствольной коробки (есть вариант для коробки с ручкой для переноса), ствольной накладки, нижней части ствольной коробки, ствола, приклада и других деталей. Есть также отсек для инструментов, которые нужны для сборки. Стрелок сможет визуально определить, все ли детали он получил — если какой-то отсек не заполнен, значит комплект не полон.
